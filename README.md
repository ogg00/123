# Random Task Generator

**Автор:** Артёмов Егор Олегович
## Описание
Random Task Generator — это GUI-приложение на Python (Tkinter), которое генерирует случайные задачи из предопределённого списка, хранит историю и позволяет фильтровать её по типу задачи. История сохраняется и загружается через JSON.

## Возможности
- Случайная генерация задач с помощью `random.choice`.
- Добавление новых задач с проверкой на пустую строку.
- Отображение истории в виде списка/таблицы.
- Фильтрация по типу задачи: учёба, спорт, работа.
- Сохранение и загрузка данных в JSON.

## Как использовать
1. Установите Python 3.
2. Запустите файл `task_generator.py`.
3. При необходимости добавьте новую задачу и выберите её тип.
4. Нажмите «Сгенерировать задачу».
5. Используйте фильтр по типу, чтобы отобрать нужные записи.
6. Сохраняйте или загружайте историю через JSON-кнопки.

## Примеры тестов
- Добавить пустую задачу: программа должна показать ошибку.
- Сгенерировать несколько задач: они должны появиться в истории.
- Применить фильтр «Спорт»: в таблице должны остаться только задачи этого типа.
- Сохранить историю в JSON и загрузить её обратно: список должен восстановиться.
